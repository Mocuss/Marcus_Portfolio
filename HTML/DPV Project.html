Singapore Tourism Analysis

This project was a group project.

Due to Covid 19 restrictions, Tourism had severely declined back in 2020.
Although the sector has recovered, it has not reached its former state.
Hence, our goal was to to find insights that can assist the growth of Singapore's tourism.

As the group leader, my role was to:

  • Look for data and create data dictionaries
  • Create or/and assist in the creation of the 3 dashboards
  • Do Analysis and Insights from the dashboards


The software used in this project were:

  1. Google Colab/Visual Studio Code to clean the data.
  2. Power BI for dashboards
  3. Excel to store the data

Process

Data Mining Goals:

Our dashboards fulfilled the 3 data mining goals, which were:


  1. Tourist Demographics affecting tourism
  2. Identify top tourism receipts
  3. Identify if revenge tourism is viable

Dashboards:

After my teammates cleaned the data by removing duplicates, missing data and renaming columns, I created the dashboards.

Dashboard 1: Tourist Demographics

Dashboard info:

  • Pie charts were used to show distributions between gender of tourists
  • Line Graphs for country of arrival and age of tourists
  • Bar graphs for when the tourists arrive
  • Filters such as year filters and quarter filters were added to view any insights easily.
  • Key performance indicators were also used to view average total demographics.

The following are the other 2 dashboards that I was heavily involved in creating.

Dashboard 2: Tourist Spendings

Dashboard 3: Revenge Tourism

Analysis and Insights

The following are examples of insights from dashboard 1:


Majority of the tourists coming to Singapore are working adults (25-54 Y.O.). This means Singapore could focus on MICE tourism or catered towards working adults with disposable income.


Tourists often arrive during the later half of the year, such as July (1.17m arrivals annually) and December (1.15 million arrivals annually), meaning we can create promotions during that period to attract more tourists.
